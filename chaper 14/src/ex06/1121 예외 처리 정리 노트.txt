예외 처리(Exception Handler)
1. 방법
    가.직접 처리(try-catch로 감싸기)
    나.예외 전가(throws 사용)
2. try-catch
    가. try()
        -() 안에 : 자동으로 리소스를 반납해야하는 클래스를 기입(ex.stream)
        -{} 안에 : 예외 발생 가능성이 있는 코드들을 기입
    나. catch(): try{}에서 예외가 발생했을 시 처리하는 차선책을 적는 곳


3. 빨간 줄이 에러인지 예외인지 구분하는 방법

->커서를 가져다 대었을 때 if(execption 글자 존재) -> 예외, else -> 에러

4. 직접 예외 처리하기 좋은 방법과 throws하기 좋은 방법
-> 하나의 메서드 안에 처리해야 할 예외가 두 개 이상이면 ->throws, 아니면 직접 예외 처리
