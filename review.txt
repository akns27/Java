//필드와 메서드 앞에 올 수 있는 4가지 '접근 지정자'
필드와 메서드 앞에 오는 것
public : 어디서나 사용가능
protected : 같은 패키지 + 다른 패키지의 자식 클래스
default : 같은 패키지에서만
private :  같은 패키지 같은 파일에서만

public과 private가 주로 쓰임

클래스 앞에 오는 것
가. public : 다른 클래스에서 해당 클래스를 부를 수 있다.
나. default :다른 클래스에서 import를 할 수 없다.
